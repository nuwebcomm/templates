Tables

Tables are automatically wrapped in a "responsive-table" div for mobile scrolling with default table row stripe and table row hover options.

Multiple classes can be used on the same <table> to achieve your desired outcome. You can use following classes:

  • <table class="table-borders"> - adds a light grey border around table and cell
  • <table class="table-no-stripe"> - removes alternating grey row stripes
  • <table class="table-no-hover"> - removes purple table row hover state
  • <table class="table-center-header"> - centers all text within <th>
