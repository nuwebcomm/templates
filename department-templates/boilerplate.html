This page will show the various styles associated with a standard content page.

The default link style is bold and purple with a dashed bottom border. If a dashed bottom border is undesireable (i.e. an anchor around an image), use <a href="#" class="no-dash">.

You can also apply an arrow link style by using: <p class="arrow-link"><a href="#">Arrow Link</a></p>. This also works on spans and anchors but anchor classes have less desireable results.

[TABLES]

The standard table style is borderless with a purple table head, alternating row stripes and a row hover state for each row. Additionally, you can add a "table-borders" class to apply a light grey border around the table and cells. You can also add "table-no-stripe" to remove the alternating stripes and "table-no-hover" to remove the hover state with each table.
